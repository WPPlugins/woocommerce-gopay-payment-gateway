== Description ==

Tento plugin přidá do **WooCommerce** platební bránu **GoPay**.

= Funkce: =
* testovací/produkční prostředí GoPay
* automatické platby
* HTTP notifikace
* změna popisu/názvu platby
* automatická změna stavu objednávky
* paymentSessinID v poznámkách při změně stavu objednávky

= Stavy objednávek: =
* Čeká na vyřízení - nebyla vybrána platební metoda
* Selhalo - platba byla zrušena/chyba během platby
* Čeká na vyřízení - platba čeká na provedení
* Zpracovává se, Dokončeno - platba proběhla

== Installation ==

= Automatická instalace =
V administraci Wordpress vyhledejte tento plugin, nainstalujte ho a dejte aktivovat.

= Ruční instalace =
Stáhněte si a rozbalte Zip archív pluginu a poté jej nahrajte do složky /wp-content/plugins. A aktivujte ho v administraci Wordpress.
